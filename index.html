Code Intelligence Protocol

Standardizing how tools describe code.

SCIP (pronounced “skip”) is a language-agnostic protocol that standardizes how indexers emit code intelligence data like Go to definition, Find references, and Find implementations.

Install the CLI
At a glance
  • Standardized index format
  • Language-agnostic schema for indexers
  • Interoperable tools and integrations
  • CLI utilities for working with indexes

Indexers and tooling using SCIP

SCIP's standard index format is already emitted by a growing ecosystem of indexers across major languages.

  • C#, Visual Basic
  • C++, C
  • Dart
  • Go
  • Java, Scala, Kotlin
  • PHP
  • Python
  • Ruby
  • Rust
  • TypeScript, JavaScript

Install the SCIP CLI

Grab prebuilt binaries from the SCIP releases page, or compile locally from source. Check the CLI reference or use --help for usage.

Download releases CLI reference 
git clone https://github.com/sourcegraph/scip.git --depth=1
cd scip
go build ./cmd/scip
